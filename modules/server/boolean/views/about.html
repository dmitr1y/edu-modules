Замкнутые классы булевых функций

T0

Класс функций, сохраняющих константу 0:

T1

Класс функций, сохраняющих константу 1:

S

Класс самодвойственных функций:

M

Класс монотонных булевых функций:

L

Класс линейных булевых функций:

U

Класс функций одной переменной U, содержащий только константы, отрицание и селектор (функцию, равную одному из своих аргументов на всех наборах их значений).

C

Через С обозначается множество всех функций, которые не имеют существенных переменных (т.е. равных константам).

P2

Множество P2 всех возможных булевых функций замкнуто. Любой замкнутый класс булевых функций, отличный от P2, целиком содержится хотя бы в одном из пяти предполных классов (T0, T1, L, M, S).

I∞

Класс I∞ функций, для которых выполнено условие , где xi — одна из переменных функции.

O∞

Класс O∞ функций, для которых выполнено условие , где xi — одна из переменных функции.

Om

Класс Om функций (m — любое натуральное, большее единицы число), удовлетворяющих следующему условию: для любых m наборов, на которых функция принимает нулевое значение, найдется переменная, также принимающая нулевое значение на всех этих наборах.

Im

Класс Im функций (m — любое натуральное, большее единицы число), удовлетворяющих следующему условию: для любых m наборов, на которых функция принимает единичное значение, найдется переменная, также принимающая единичное значение на всех этих наборах.

D

Класс дизъюнкций D, являющийся замыканием множества . Он представляет собой множество функций вида .

K

Класс конъюнкций K, являющийся замыканием множества . Он представляет собой множество функций вида .
