Описание модуля

Модель поведения муравьев, основанную на муравьином алгоритме

Системные требования

  • Операционная система: Windows
  • Необходимые пакеты: Microsoft Visual Redistributable

Инструкции

  1. Запустите программу
  2. Теперь вы можете наблюдать за поведением муравьев
  3. Далее для удобного просмотра вы можете нажать кнопку 1 и использовать следующие кнопки:
    • W-выше
    • S-ниже
    • A-левее
    • D-правее
    • O-увеличение
    • P-уменьшение
    • F-добавить еду
  1. Запустите программу
  2. Нажмите Q для добавления еды на поле симулятора
  3. Теперь вы можете наблюдать за поведением муравьев
